# TugasPCD
Cara Menjalankan Source Code
1. Menjalankan di Google Colab
Langkah-langkah

- Buka Google Colab di
https://colab.research.google.com

- Pilih menu File → Upload Notebook.
Upload file TugasPCD2.ipynb.

  Notebook akan terbuka di Colab.

- Jalankan seluruh kode dengan Runtime → Run all
atau jalankan setiap cell dengan tombol Run.

- Jika muncul error "module not found", install dependency berikut:

  !pip install opencv-python numpy matplotlib

2. Menjalankan di Laptop Menggunakan Jupyter Notebook

- Install Python dan Jupyter Notebook

Jika belum ada:

- buka Terminal di Laptop/PC kamu Lalu Ketik:

  pip install notebook

  - Menjalankan Jupyter

    jupyter notebook

- Langkah berikutnya

   - Browser akan terbuka.

   - Pilih file TugasPCD2.ipynb.

   - Jalankan cell satu per satu melalui tombol Run.

   - Jika ada library yang belum terinstall, jalankan:

  pip install numpy opencv-python matplotlib

3. Menjalankan di VS Code

Persiapan

- Install ekstensi:

a. Python

b. Jupyter

- Langkah-langkah

a. Buka file TugasPCD2.ipynb di VS Code.

b. Pastikan kernel Python sudah dipilih.

c. Klik Run All untuk menjalankan semua cell.
